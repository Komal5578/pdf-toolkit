## 🛠️ Tools Included

| Tool | Description |
|------|-------------|
| 🗜️ PDF Compressor | Reduce PDF file size |
| 🖼️ Image to PDF | Convert JPG/PNG images to PDF |
| 📸 PDF to Images | Extract each page as JPG/PNG |
| 🔗 Merge PDFs | Combine multiple PDFs into one |
| ✂️ Split PDF | Extract specific pages |
| 📝 Word to PDF | Convert DOCX documents to PDF |

## 🚀 Tech Stack

- **React + Vite** (frontend framework)
- **pdf-lib** — PDF merge, split, compress
- **pdfjs-dist** — PDF to images rendering
- **jsPDF** — Image to PDF conversion
- **mammoth.js** — DOCX parsing
- **JSZip** — ZIP packaging for image exports

## 🔒 Privacy First

All processing happens **100% in your browser**. Files are never uploaded to any server.

## 🏃 Run Locally

```bash
npm install
npm run dev
```

## 🚢 Deploy to Vercel

```bash
npm run build
vercel --prod
```

Or connect your GitHub repo to [vercel.com](https://vercel.com) for automatic deployments.
